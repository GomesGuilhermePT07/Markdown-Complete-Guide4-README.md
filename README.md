<h2 align='center'>🧱 1. Fundamentals</h2>

## 1.1 Titles (`#`)

```Markdown
# Título 1
## Título 2
### Título 3
#### Título 4
##### Título 5
###### Título 6
```

# Título 1
## Título 2
### Título 3
#### Título 4
##### Título 5
###### Título 6

## 1.2 ✍️ Emphasis

```Markdown
*italic* ou _italic_  
**bold** ou __bold__  
***bold + italic***  
~~erased~~
```

*italic* ou _italic_  
**bold** ou __bold__  
***bold + italic***  
~~erased~~

## 1.3 🗒️ Lists
### Unordered
```Markdown
- Item
* Item
+ Item
```

- Item
* Item
+ Item

### Ordered
```Markdown
1. Item
2. Item 2
   1. Subitem
```

1. Um  
2. Dois  
   1. Subitem
  
### Checkboxes
```Markdown
- [x] Done
- [ ] To do
```

- [x] Done
- [ ] To do

> NOTE: The "x" must be put only in the tasks you've already done

## 1.4 🔗 Links
```Markdown
[Google](https://google.com)
```

[Google](https://google.com)

## 1.5 🖼️ Images
```Markdown
![Alt text](https://via.placeholder.com/100)
```

![Alt text](https://via.placeholder.com/100)

## 1.6 💻 Code

### Inline
```Markdown
Use `npm install`
```

Use `npm install`

### Code block
<pre>
```
bloco de código
```
</pre>

How it will be shown:
```
bloco de código
```

### With language (Python for example)
<pre>
```Python
def ola():
    print("Olá!")
```
</pre>

How it will be shown:
```Python
def ola():
    print("Olá!")
```

<h2 align='center'>📐 2. Structure and Layout</h2>

## 2.1 📊 Tables
```Markdown
| Name  | Age | Role     |
|-------|------:|-----------|
| Name 1  | 24    | Backend developer |
| Name 2 | 32    | Fullstack developer   |
```

| Name  | Age | Role     |
|-------|------:|-----------|
| Name 1  | 24    | Backend developer |
| Name 2 | 32    | Fullstack developer   |

> IMPORTANT
>> `:---:` -> aligned to the center <br> `---:` -> aligned to the right <br> `:---` -> left aligned

## 2.2 ➖ Horizontal line
> All 3 do the same thing
```Markdown
___
---
***
```

___
---
***

## 2.3 💬 Quotes
```Markdown
> This is a quote.
>> Subquote
```

> This is a quote.
>> Subquote

<h2 align='center'>❌ 5. Bad habits to avoid</h2>

 - ❌ Titles with `**bold**`
 - ❌ Mix ordered with unordered lists
 - ❌ Using advanced HTML inside a Markdown document
 - ❌ Links without proper text
